# React-blogify

This is a blog like app where I develop only front-end. Technology used, React, Redux-toolkit, Rtk query.


## Installation

Install my-project with npm/yarn

```bash
 clone this repo:
 npm install, 
 cd api and run: npm run dev,
 cd front-end run: npm run dev
```
